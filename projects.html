Hardware Tasks

    555-Astable Multivibrator
    I designed a 555 astable multivibrator with duty cycle 60%, by building the circuit on a breadboard and by using the probes, observed the output of my circuit on the DSO.
    Soldering
    I learnt about the soldering equipments like the solder, the soldering iron, soldering wick, flux, etc. I lLearnt to use them and perform basic soldering on a perf board.
    LED Toggle Using ESP32
    I built the circuit using an ESP32 that controls the LED's connected to the GPIOs. Later, I used the arduino IDE to code and upload my program to the ESP32. I created a standalone web server, that further, controlled the ESP32.
    Speed control of a DC Motor
    I explored the basic techniques for controlling DC Motors using the L298N motor driver and the Arduino board. Using an UNO and H-Bridge L298N motor driver, I could control the speed of a 5V BO motor, simulated it on tinkercad and performed it on hardware.
